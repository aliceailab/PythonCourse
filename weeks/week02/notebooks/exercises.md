# Week 2 Exercises

Try these in your notebook or in a Python file.

## Part A: Coordinate Warm-Up

1. Print the coordinate pair `(2, 5)`.
2. Store `x = 4` and compute `y = 2 * x + 1`.
3. For `x = -3`, compute `y = x ** 2`.
4. Write a loop that prints coordinate pairs for `x` from 0 to 5 using `y = x + 2`.

## Part B: Tables and Patterns

5. Print a table of `x` and `x ** 2` for `x` from 1 to 10.
6. Print a table of `x`, `x ** 2`, and `x ** 3` for `x` from 1 to 10.
7. Use a nested loop to print this pattern:

```text
*
**
***
****
*****
```

8. Change the pattern so it prints numbers instead of stars.

## Part C: Functions for Equations

9. Write a function `line(x)` that returns `2 * x + 1`.
10. Write a function `parabola(x)` that returns `x ** 2`.
11. Write a function `shifted_parabola(x)` that returns `(x - 2) ** 2 + 1`.
12. Write a function `distance_from_origin(x, y)` that returns the distance from `(0, 0)` to `(x, y)`.

## Part D: Lists of Points

13. Build a list of points for `y = 3x - 2` for `x` from `-5` to `5`.
14. Build a list of points for `y = x ** 2` for `x` from `-5` to `5`.
15. Print the x-values and y-values as separate lists.

## Part E: First Plotting Practice

16. Plot a line using `matplotlib`.
17. Plot a parabola using `matplotlib`.
18. Add a title and x-axis and y-axis labels.
19. Plot both a line and a parabola on the same graph.

## Stretch Challenges

20. Build x-values from `-10` to `10` and plot `y = x ** 2 - 4`.
21. Make a rough circle using:
