# Week 2: Problem Solving and Visualization

This week is about moving from basic Python into visual and structured problem solving.

You will practice:

- breaking problems into smaller steps
- designing helper functions
- working with coordinates
- building lists of points
- using loops to generate patterns
- making your first mathematical plots

The goal is to make Python feel more visual, playful, and mathematical.

## What You Will Be Able to Do

By the end of Week 2, you should be able to:

- break a problem into smaller coding steps
- generate coordinates with Python
- build x-values and y-values for simple equations
- use loops to create patterns and tables
- understand the idea of plotting a line, parabola, and circle
- build a small graphing playground script

## How To Study This Week

Use 4 study days. Aim for 60 to 90 minutes each day.

### Day 1: Problem Solving with Small Steps

Learn:

- how to break a task into inputs, process, and output
- how helper functions make code easier to test
- how loops help generate many values at once

Practice:

- build a table of `x` and `x ** 2`
- write a function for `y = 2x + 1`
- print coordinate pairs like `(x, y)`

### Day 2: Coordinates and Patterns

Learn:

- x-axis and y-axis ideas
- ordered pairs
- using loops to generate points
- how a line or parabola becomes a list of coordinates

Practice:

- generate points for a line
- generate points for a parabola
- print a simple number pattern with nested loops

### Day 3: First Graphs in Python

Learn:

- basic `matplotlib` plotting
- titles and labels
- plotting multiple points
- comparing equations visually

Practice:

- plot a line
- plot a parabola
- plot a rough circle from coordinate formulas

### Day 4: Mini-Project

Build and test:

- `weeks/week02/project/graphing_playground.py`

Your playground should generate and plot simple math objects like:

- a line
- a parabola
- a circle

```bash

```
