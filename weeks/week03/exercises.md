# Week 3 Exercises

Try these in your notebook or in a Python file.

## Part A: Divisibility

1. Check whether 12 is divisible by 3.
2. Check whether 17 is divisible by 5.
3. Write a function `is_divisible(a, b)` that returns `True` if `a` is divisible by `b`.
4. Print all numbers from 1 to 30 that are divisible by 4.

## Part B: Factors

5. Write a function `factors(n)` that returns the positive factors of `n`.
6. Find the factors of 12, 18, 24, and 30.
7. Write a function `factor_count(n)` that returns how many factors `n` has.

## Part C: GCD and LCM

8. Write a function `gcd(a, b)` using a loop.
9. Find `gcd(12, 18)`.
10. Find `gcd(20, 30)`.
11. Write a function `lcm(a, b)`.
12. Find `lcm(6, 8)` and `lcm(12, 18)`.

## Part D: Prime Numbers

13. Write a function `is_prime(n)`.
14. Test `is_prime()` on 2, 3, 4, 11, and 15.
15. Write a function `primes_up_to(n)` that returns all primes from 2 to `n`.
16. Find all primes up to 50.

## Part E: Algebra

17. Write a function `linear(x)` that returns `2 * x + 3`.
18. Find the value of `linear(4)`.
19. Find which integer value of `x` makes `2 * x + 3 == 11`.
20. Write a loop to test values of `x` from `-10` to `10` for `x ** 2 - 5 * x + 6 == 0`.

## Part F: Optional Sympy

21. Use `sympy` to solve `2 * x + 3 = 11`.
22. Use `sympy` to solve `x ** 2 - 5 * x + 6 = 0`.
23. Factor `x ** 2 - 5 * x + 6` with `sympy`.
