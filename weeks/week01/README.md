# Week 1: Python Foundations for Math

This week is about getting comfortable with the core building blocks of Python:

- variables
- numbers and strings
- lists
- loops
- conditionals
- functions
- simple debugging

The goal is not speed. The goal is to make Python feel natural.

## What You Will Be Able to Do

By the end of Week 1, you should be able to:

- write and run simple Python scripts
- store values in variables
- use `if`, `for`, and `while`
- create functions that take inputs and return outputs
- solve small math problems with code
- build a tiny reusable math toolbox

## How To Study This Week

Use 4 study days. Aim for 60 to 90 minutes each day.

### Day 1: Core Python Basics

Learn:

- `print()`
- variables
- integers and floats
- strings
- basic arithmetic

Practice:

- compute area of a rectangle
- convert minutes to hours
- calculate average of 3 numbers

### Day 2: Logic and Loops

Learn:

- `if`, `elif`, `else`
- comparison operators
- `for` loops
- `while` loops
- lists

Practice:

- print numbers from 1 to 10
- print even numbers from 1 to 20
- sum the first 10 integers
- check whether a number is positive, negative, or zero

### Day 3: Functions

Learn:

- how to define a function with `def`
- parameters
- return values
- why functions help organize code

Practice:

- write a square function
- write a cube function
- write a function to compute distance traveled from speed and time
- write a function to compute the average of a list of numbers

### Day 4: Mini-Project

Build and test:

- `weeks/week01/project/math_toolbox.py`

Your toolbox should contain small functions for common math tasks.

## How To Run Python Files

From the repo root:

```bash
python3 weeks/week01/project/math_toolbox.py
```

If you want to experiment quickly, you can also open Python in the terminal:

```bash
python3
```

Then try:

```python
print(2 + 3)
print(7 * 8)
```

## Week 1 Checklist

- I can run a Python script.
- I can use variables to store values.
- I can write an `if` statement.
- I can write a `for` loop.
- I can create and call a function.
- I can solve small math problems in Python.
- I can read and edit `math_toolbox.py`.
