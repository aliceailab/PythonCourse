# Week 1 Exercises

Try these in your own Python file or in the Python terminal.

## Part A: Warm-Up

1. Print your name.
2. Print your favorite number.
3. Store two numbers in variables and print their sum.
4. Compute the area of a triangle using `0.5 * base * height`.
5. Convert 90 minutes into hours.

## Part B: Conditionals

6. Write a program that checks whether a number is even or odd.
7. Write a program that checks whether a number is positive, negative, or zero.
8. Given a test score, print:

- `"A"` if score >= 90
- `"B"` if score >= 80
- `"C"` if score >= 70
- `"D"` if score >= 60
- `"F"` otherwise

## Part C: Loops

9. Print the integers from 1 to 20.
10. Print only the multiples of 3 from 1 to 30.
11. Compute the sum of the integers from 1 to 100.
12. Compute the product of the integers from 1 to 5.

## Part D: Functions

13. Write a function `square(x)` that returns `x * x`.
14. Write a function `cube(x)` that returns `x * x * x`.
15. Write a function `is_multiple(a, b)` that returns `True` if `a` is a multiple of `b`.
16. Write a function `average_of_three(a, b, c)` that returns the average.

## Part E: Small Math Challenges

17. Write a function to convert Celsius to Fahrenheit.
18. Write a function to compute the perimeter of a rectangle.
19. Write a function to compute the distance traveled at constant speed.
20. Write a function that returns the larger of two numbers without using `max()`.

## Stretch Challenges

21. Write a function `factor_count(n)` that returns how many positive factors `n` has.
22. Write a function `sum_of_squares(n)` that returns:

```python
1**2 + 2**2 + 3**2 + ... + n**2
```

23. Write a function `is_prime(n)` for positive integers greater than 1.
