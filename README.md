# PythonCourse

A beginner-friendly Python learning repo focused on math, probability, statistics, data science, machine learning, and AI.

This repository is designed as a practical, project-based path for building Python skills from a very basic starting point. The emphasis is on learning by coding, visualizing ideas, and building small projects step by step.

## What This Repo Covers

- Python foundations
- algebra and number theory
- geometry and calculus
- linear algebra
- probability and statistics
- data science with `pandas`
- machine learning with `scikit-learn`
- AI and neural network foundations
