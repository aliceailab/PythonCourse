# Week 1 Summary

This week focused on Python foundations for math. By completing the Week 1 workout notebook and the `math_toolbox.py` mini-project, you built the first layer of practical Python skill.

## What I Learned

### 1. Running Python and Reading Output

You practiced running Python code and using `print()` to see results. This is the basic feedback loop of programming:

- write code
- run it
- inspect the output
- change it and try again

### 2. Arithmetic in Python

You used Python as a calculator for:

- addition
- subtraction
- multiplication
- division
- powers

You also used arithmetic for simple math problems such as area, averages, distance, and temperature conversion.

### 3. Variables

You learned how to store values in variables and reuse them in calculations.

Examples:

- width and height
- temperature values
- favorite number
- intermediate results like area or average

This is important because variables let your code become flexible instead of hard-coded.

### 4. Conditionals

You practiced using:

- `if`
- `elif`
- `else`

This means you can now write programs that make decisions, such as:

- checking whether a number is positive, negative, or zero
- checking whether a number is even or odd
- assigning a letter grade from a score

### 5. Loops

You used:

- `for` loops
- `while` loops

This means you can now repeat work in code instead of writing the same line again and again.

Examples included:

- counting from 1 to 10
- printing even numbers
- summing integers
- making a countdown

### 6. Lists

You worked with lists to store multiple numbers together. You practiced:

- reading items from a list
- finding the length of a list
- looping through a list
- computing totals and averages from list values

This is an important step toward data analysis later.

### 7. Functions

You learned how to define your own functions using `def` and `return`.

This means you can now create reusable tools instead of rewriting logic each time.

Examples from Week 1 included functions for:

- squaring and cubing numbers
- checking whether a number is even
- finding triangle area
- averaging values
- computing distance

### 8. Building a Mini Math Toolbox

In `math_toolbox.py`, you combined several skills into a small project. That is a big step because you were no longer just trying isolated commands. You were organizing useful functions into a single Python file.

Your toolbox included ideas such as:

- square and cube
- average
- even-number test
- rectangle and triangle area
- distance
- Celsius to Fahrenheit conversion
- factor counting
- prime checking

This is your first real coding project in the repo.

## What I Can Do Now

After Week 1, you should be able to:

- run a Python script
- edit a Python file
- use variables in math calculations
- write simple `if` statements
- write basic `for` and `while` loops
- work with lists of numbers
- define and call functions
- build small math utilities in Python

## Why This Matters

These are the core building blocks for everything that comes next:

- algebra coding
- number theory functions
- graphing and geometry
- statistics and simulations
- data science
- machine learning

Without this foundation, later topics feel confusing. With this foundation, later topics become much easier to learn.

You have made real progress if:

- Python code looks less intimidating than before
- you can predict what simple code will do before running it
- you can fix small mistakes by yourself
- you can turn a math idea into a short function
