# Python Learning Course for Math, Data Science, Machine Learning, and AI

## Goal

Build practical Python skills for:

- algebra
- number theory
- geometry
- calculus
- linear algebra
- probability and statistics
- data science
- machine learning
- AI foundations

This course is designed for a learner with experience in Scratch and very basic Python and JavaScript drawing. The focus is on learning by coding, visualizing ideas, and building small projects.

## Course Structure

- Length: 10 weeks
- Pace: 4 days per week
- Time: 60 to 90 minutes per study day
- Style: project-based and hands-on

## Learning Outcomes

By the end of this course, you should be able to:

- write clean Python for mathematical and data problems
- use Jupyter notebooks for experiments and explanations
- solve and visualize problems with `numpy`, `sympy`, and `matplotlib`
- work with datasets using `pandas`
- build beginner machine learning models with `scikit-learn`
- understand the core ideas behind neural networks and modern AI

## Weekly Plan

### Week 1: Python Foundations for Math

Topics:

- variables and data types
- expressions and operators
- lists and loops
- conditionals
- functions
- printing and debugging

Practice:

- arithmetic exercises
- sequences and patterns
- unit conversion problems
- simple math puzzles

Mini-project:

- Build a `math_toolbox` with functions for powers, roots, factors, averages, and basic equation helpers.

### Week 2: Problem Solving and Visualization

Topics:

- function design
- nested loops
- breaking problems into steps
- plotting with `matplotlib`

Practice:

- graph points and lines
- draw parabolas and circles
- explore coordinates and slopes

Mini-project:

- Build a graphing playground for algebra and geometry.

### Week 3: Algebra and Number Theory with Python

Topics:

- solving simple equations
- factoring
- divisibility
- greatest common divisor and least common multiple
- modular arithmetic
- prime numbers

Tools:

- `sympy`

Practice:

- generate primes
- test divisibility
- factor integers
- solve symbolic equations

Mini-project:

- Build an equation solver and prime/factorization explorer.

### Week 4: Geometry and Calculus

Topics:

- distance and midpoint
- slope and angle
- vectors in 2D
- function graphs
- derivatives and tangent lines
- definite integrals and area

Tools:

- `sympy`
- `matplotlib`

Practice:

- plot functions
- compare a function and its derivative
- estimate and visualize area under a curve

Mini-project:

- Create an interactive calculus notebook that shows a function, slope, and area.

### Week 5: Linear Algebra for Coding

Topics:

- vectors
- matrices
- matrix multiplication
- systems of equations
- transformations in 2D
- eigenvalue intuition

Tools:

- `numpy`

Practice:

- solve systems numerically
- transform points with matrices
- visualize scaling, rotation, and reflection

Mini-project:

- Build a visual demo of 2D linear transformations.

### Week 6: Probability and Statistics

Topics:

- randomness and simulation
- discrete probability
- expectation and variance
- mean, median, and standard deviation
- sampling
- correlation

Tools:

- `numpy`
- `matplotlib`

Practice:

- simulate coin flips, dice rolls, and card draws
- compare theoretical and experimental probability
- graph distributions and histograms

Mini-project:

- Build a probability simulator with charts and summary statistics.

### Week 7: Data Science Foundations

Topics:

- loading CSV data
- cleaning missing or messy values
- filtering and sorting
- grouping and aggregation
- exploratory data analysis

Tools:

- `pandas`
- `matplotlib`

Practice:

- inspect a real dataset
- answer simple questions from data
- create clear visual summaries

Mini-project:

- Analyze a real dataset and write five evidence-based conclusions.

### Week 8: Machine Learning Basics

Topics:

- supervised learning
- train/test split
- overfitting
- evaluation metrics
- regression and classification

Tools:

- `scikit-learn`

Practice:

- linear regression
- logistic regression
- decision trees

Mini-project:

- Build a model to predict or classify a simple dataset, then explain the results in plain language.

### Week 9: Intro to AI and Neural Networks

Topics:

- what a neural network is
- features, weights, loss, and training
- tensors and layers
- model predictions

Tools:

- `pytorch` or `tensorflow` at a beginner level

Practice:

- train a tiny neural network
- compare predictions before and after training

Mini-project:

- Create a simple digit classifier or image classifier demo.

### Week 10: Capstone Project

Choose one track:

- Math track: symbolic solver and visual explorer
- Data track: full data analysis notebook
- ML track: end-to-end prediction project
- AI track: beginner neural network demo

Final deliverable:

- one polished project
- a notebook or script with clear code
- charts or outputs
- short written explanation of what you built and learned

## Suggested Weekly Routine

### Day 1

- Learn new concepts.
- Read examples and run them yourself.

### Day 2

- Practice coding exercises.
- Rewrite solutions into clean functions.

### Day 3

- Work on a mini-project.
- Add simple plots or outputs.

### Day 4

- Review mistakes.
- Improve code style.
- Write short notes about what you learned.

## Study Advice

- Type every example yourself instead of only reading.
- Keep one notebook or folder per topic.
- Focus on small projects every week.
- Visualize ideas whenever possible.
- Explain your code in plain English after each project.
- Revisit earlier work and improve it as your skills grow.
